Simple Jquery Lightbox Template

After coding my website from scratch, I wanted to add additional functionality by having each image appear in a lightbox and have the screen dim to make the image really “pop”.

This was as much of an exercise in basic javascript and coding skills, as it was an exercise in problem solving and communication.
After looking around for a solution, I noticed there were few for people not already using a content management system like tumblr or wordpress which come with this feature right out of the box. I coded this entire lightbox image gallery from scratch, by hand, all without using any tutorials or templates on my own. This was as much of an exercise in basic javascript and coding skills, as it was an exercise in problem solving and communication. I coded this app to be completely responsive, and resize/adapt to any device. This was an undertaking that took some logic to accomplish. Lastly, I filled the gallery with my own photography, to help advertise an additional skill. The code is open source, and users are free to download it and use it however they please. I’ve even included instructions on its integration both inline throughout the code, as well as below the sample gallery itself.
