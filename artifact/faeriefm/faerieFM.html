FaerieFM -Rhythm Game

FaerieFM is a web based rhythm game that I'm building by myself using PixiJS (a 2D javascript rendering library), and will eventually expand it to utilize electron for PC distribution over steam and itch.io, and cocoonJS for mobile distribution. I'm also interested in utilizing decentralized web technology, Blockstack, for both distribution of the game as well as management of user created content that they themselves host and serve to others over the blockstack network. Make no mistake, this project has been both a labor of love and a massive undertaking. Not only have I coded this game from scratch, but have also made every visual asset and piece of artwork entirely by myself.

I've worked diligently on this project for over 2 years now, and intend to see it through to completion...

Being a rhythm based, the core systems for the game are more complicated than meets the eye. Every movement has to be tracked and kept on beat with the music. In order to achieve this, I had to come up with a mathematical equation that tracks beats and converts said beats to pixel movements on the device being rendered to. For the art, the lovable faerie mascot, Musette, I designed using silhouetting and other design principles I learned in college. In addition, her animations I drew frame by frame, all by hand. There is far more left to do with FaerieFM, and I've only begun to scratch the surface. I've worked diligently on this project for over 2 years now, and intend to see it through to completion this year. Please look forward to it.
