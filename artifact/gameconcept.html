Interactive Game Design Document

I created this website to make a lengthy game design document more presentable for potential backers and collaborators. The main challenge was working with unformatted walls of text and styling them for readability. The website is a 1-page design with a scrolling navbar. I started by tagging and styling the document with HTML/CSS, used wordpress to streamline the process, and altered the theme’s code I used to fit my needs. I broke up the document into sections, and further broke each section into readable blocks of text with an optimum line length of 70-80 characters. Lastly, I added drop caps and my concept illustrations for additional entry points, choosing a simple typeface, Lato, for the body text.

The contents of the document, including the illustrations, were all created and designed by me. “Faerie Heart” is my original concept.
