Portfolio Open House Postcard Design

Iwas tasked with designing a mockup for an open house event at my college. For this design, I wanted to approach it by using as little as I could get away with as a designer, but at the same time make it eye-catching and fun. Using Kennesaw State University’s style guide, I utilized the school colors and created a fun yet minimalist circle design to help guide the viewer’s eye. Elements on the page are arranged in a hierarchy that takes the viewers eye from the top left to the bottom right diagonally. I used basic visual icons to help give the viewer a quick idea of what types of creative disciplines will be represented at the event. The main challenge to this design problem was working under a strict set of guidelines with a very strict set of information.
